h2. Фоновые изображения

*Цель работы:* ознакомиться со способами создания стилевых описаний. Получить навык манипуляции параметрами фоновых изображений 

Откройте файл simpsons.html и ознакомьтесь с "скриншотом simpsons.png":http://projects.avalon.ru/attachments/download/582/simpsons.png 

Нажмите кнопку «New CSS Rule» на палитре «Css Styles». Возможно потребуется развернуть палитру 

!new-rule.png! 

В открывшемся окне укажите: 

* Selector Type: class 
* Selector Name: bart 
* Rule Definition: simpsons.css 

Нажмите кнопку «ОК» 

В открывшемся окне укажите: 

* Background 
** Background-image: выберите изображение Барта 
** Background-repeat: no-repeat 
** Background-position (X): center 
** Background-position (Y): center 
* Box 
** height: 1200px 

Нажмите кнопку «ОК» 

Аналогичным образом создайте стилевые описания для классов: «lisa», «homer», «maggie», «marge». Для указанных элементов укажите только параметры фонового изображения

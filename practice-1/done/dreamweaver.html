Обзор

Дизайн, разработка и сопровождение веб-сайтов и приложений на базе отраслевых стандартов

Программное обеспечение Adobe® Dreamweaver® CS5 позволяет дизайнерам и разработчикам создавать веб-сайты профессионального уровня на базе отраслевых стандартов. Поддержка визуального дизайна и изменения кода, разработка страниц для систем управления контентом, а также точность тестирования совместимости с браузерами благодаря интеграции с Adobe BrowserLab, новой онлайн-службой Adobe CS Live**.В течение ограниченного периода времени услуги CS Live предоставляются бесплатно.

Визуальный дизайн сайтов

Ускоренное создание веб-страниц на базе отраслевых стандартов благодаря обновленным заготовкам макетов CSS. Проверка CSS в представлении Live View и возможность визуальной корректировки дизайна с точной настройкой полей, отступа и других параметров.

Совместимость с браузерами

Тестирование веб-страниц на совместимость с различными браузерами и операционными системами в Adobe BrowserLab*. Проверка соответствия на уровне пиксельной точности путем сравнения внешнего вида в браузере в разных режимах: по отдельности, рядом или с наложением.

Интеграция видео FLV

Импорт любых файлов FLV непосредственно в Dreamweaver и быстрое внедрение выбранной темы оформления проигрывателя. Удобство настройки параметров размера экрана, автозапуска и повтора при подготовке видеофайлов для размещения в Интернете.

Визуальный дизайн веб-сайтов

Простое создание веб-сайтов на базе отраслевых стандартов. Использование средств визуального кодирования CSS и готовых шаблонов, быстрое тестирование страниц и экономия времени благодаря интеллектуальной интеграции с приложениями Adobe® Creative Suite®.

Встроенная поддержка CMS

Поддержка создания и тестирования материалов для таких систем управления контентом, как WordPress, Joomla! и Drupal. Функция "Динамически связанные файлы" обеспечивает прямой доступ к связанным файлам страниц, в том числе и динамических, а "Навигация интерактивного просмотра" позволяет выполнять предварительный просмотр динамических приложений.

Интеграция с Adobe BrowserLab

Отображение динамических страниц и локального контента при помощи различных средств просмотра, диагностики и сравнения. Приложение Adobe Dreamweaver® поддерживает интеграцию с BrowserLab, новой онлайн-службой Adobe CS Live**, предназначенной для быстрого и точного тестирования веб-контента в различных браузерах и операционных системах.

Проверка CSS

Визуальное отображение модели CSS, а также удобные средства включения и отключения свойств CSS без необходимости чтения кода или приобретения отдельных утилит.

Заготовки макетов CSS

Создание веб-сайтов на базе отраслевых стандартов при помощи удобных автоматизированных заготовок макетов CSS. Расширенный выбор шаблонов, в которых реализованы новые наработки CSS, а также удобные селекторы и правила.

Простая настройка сайтов

Полностью переработанное диалоговое окно «Определение сайта» в Dreamweaver помогает создавать основу для сайтов еще быстрее. Добавление новых серверов с пользовательскими именами для создания статических, сетевых и других типов сайтов.

Эффективное написание кода

Создание веб-сайтов на базе отраслевых стандартов при помощи одного из ведущих в отрасли средств разработки с интеллектуальными функциями подсказки по коду и поддержкой передовых сред и технологий. Изучение новых функций и обмен ресурсами в обширном сообществе разработчиков.

Подсказка по пользовательским классам PHP

Просмотр синтаксиса пользовательских функций PHP для предупреждения возникновения ошибок при написании кода. Динамическая подсказка по коду PHP для удобства создания и сопровождения приложений, а также знакомства с важнейшими функциями, методами и объектами PHP

Поддержка Subversion

Эффективное управление файлами компонентов сайта благодаря совместной работе в среде с поддержкой контроля версий и улучшенной совместимостью с Subversion. Перенос, копирование и удаление локальных файлов — в точности как при работе с обычными статическими веб-сайтами — с возможностью синхронизации с локальным репозиторием SVN.

Подсказка по коду для отдельных сайтов

Подсказка по кодированию нестандартизированных файлов и каталогов в Adobe Dreamweaver® CS5, а также улучшенные средства подсказки по сторонним PHP-библиотекам и системам управления контентом, в том числе WordPress, Drupal и Joomla!.

Функции Dreamweaver

Ведущая в отрасли среда веб-разработки

Создание веб-сайтов на базе отраслевых стандартов при помощи ведущего в отрасли средства веб-разработки. Возможность визуального дизайна и работы с кодом, инструменты проверки CSS для повышения производительности работы, а также поддержка систем управления контентом.

Интеллектуальная поддержка написания кода

Быстрое написание оптимизированного кода. Подсказки по кодированию в таких средах разработки HTML, JavaScript и Ajax, как Spry, jQuery и Prototype. Знакомство с важнейшими функциями, методами и объектами PHP

Полная поддержка CSS

Дизайн и разработка веб-сайтов при помощи мощных средств создания кода CSS. Визуальное отображение модели CSS без необходимости использования дополнительных утилит и редактирования кода CSS вручную даже во внешних таблицах стилей.

Поддержка ведущих технологий

Дизайн и разработка в среде, которая поддерживает большинство ведущих технологий веб-разработки, включая HTML, XHTML, CSS, XML, JavaScript, Ajax, PHP, Adobe ColdFusion® и ASP.

Расширенное сообщество Dreamweaver

Изучение новых функций и обмен ресурсами в обширном сообществе пользователей Dreamweaver. Интерактивная служба Adobe Design Center и Adobe Developer Connection, обучение и семинары, программы сертификации разработчиков и пользовательские форумы.

Системные требования

Windows

  • Процессор Intel® Pentium® 4 или AMD Athlon® 64
  • Microsoft® Windows® XP с пакетом обновления SP2 (рекомендуется пакет SP3); Windows Vista® Home Premium, Business, Ultimate или Enterprise с пакетом SP1 либо Windows 7
  • 512 Мб оперативной памяти
  • 1 Гб свободного пространства на жестком диске для установки; дополнительное свободное пространство, необходимое для установки (не устанавливается на съемные устройства хранения флэш)
  • Разрешение монитора 1280x800 с 16-разрядной видеокартой
  • Привод DVD-ROM
  • Широкополосное интернет-соединение, необходимое для онлайн-услуг

Mac OS

  • Многоядерный процессор Intel®
  • Mac OS X v10.5.7 или v10.6
  • 512 Мб оперативной памяти
  • 1,8 Гб свободного пространства на жестком диске для установки; дополнительное свободное пространство, необходимое для установки (не устанавливается на диск, для которого используется файловая система, чувствительная к регистру, или на съемные устройства хранения флэш)
  • Разрешение монитора 1280x800 с 16-разрядной видеокартой
  • Привод DVD-ROM
  • Широкополосное интернет-соединение, необходимое для онлайн-услуг

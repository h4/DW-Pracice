h2. Определение сайта в Adobe Dreamweaver

bq. Adobe Dreamweaver может работать как с отдельными html- и css-файлами, так и с каталогами, рассматривая их как целостный сайт. При определении сайта становится доступна проверка целостности ссылок, автоматическое копирование графики в каталог c сайтом, публикация сайта на сервере, работа с шаблонами

Ознакомьтесь с разделом [[Создание сайта|08_Создание_сайта]] справочной информации

h3. Определение сайта

# Выполните команду _Site > New site..._
# Укажите параметры сайта
** Site name: TL
** Local Site Folder: укажите каталог TL, расположенный внутри каталога _Практика 4_ на _Workdata_
# Нажмите Save

После определения сайта изменяется содержимое палитры Files. Каталог с сайтом становится замкнутой областью. Используя контекстное меню можно выполнять все операции по созданию и изменению файлов (копирование, переименование); таким образом все опереции по работе над сайтом выполняются внутри Dreamweaver.
